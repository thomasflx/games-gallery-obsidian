# Games gallery on obsidian
A games gallery based on the Movies gallery with minimal theme

## Setup

I recommend to first follow the [Movies database tutorial](https://minimal.guide/Guides/Create+a+movie+database) to get how things work. The setup is basically the same, only changing the IMDB key for the RAWG key and the Template.
